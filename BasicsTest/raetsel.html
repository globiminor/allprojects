Für A = 4 und b > 16 und r1 > 0

-> r1b + 16 = 8K
-> r1 < 8
und r1b/8 + 2 = K

Allgemein gilt r2b + 4 = 2IA + K2 + r1 und mit t = r1b/8
-> r2b + 4 = 8I + (t + 2)2 + r1
-> r2b + 4 = 8I + t2 + 4t + 4 + r1
-> r2b = 8I + t2 + 4t + r1

Für r1 = 1 folgt mit b = 8b'
-> 8r2b' = 8I + t2 + 4t + 1 -> t muss ungerade sein -> t = 2t' + 1 -> 8r2b' = 8I + 4t'2 + 4t' + 1 + 4t + 1 -> 8r2b' = 4x + 2, was ein Widerspruch ist, und damit gilt r1 ≠ 1. Analog zeigt sich, dass r1 ≠ 5.

Für r1 = 2 folgt mit b = 4b'
-> 4r2b' = 8I + t2 + 4t + 2 -> t muss gerade sein -> t = 2t' -> 4r2b' = 8I + 4t'2 + 4t + 2 -> 4r2b' = 4x + 2, auch ein Widerspruch, also r1 ≠ 2. Analog zeigt sich, dass r1 ≠ 6.

Für r1 = 7 folgt mit b = 8b'
-> 8r2b' = 8I + t2 + 4t + 7 -> t muss ungerade sein -> t = 2t' + 1 -> 8r2b' = 8I + 4t'2 + 4t' + 1 + 8t' + 4 + 7 -> 8r2b' = 8I + 4t'(t' + 3) + 12.
t' ist entweder gerade oder ungerade, in beiden Fällen ist t'(t' + 3) gerade, und modulo 8 zu einem Widerspruch führt, also r1 ≠ 7.

Für r1 = 3 folgt mit b = 8b'
-> 8r2b' = 8I + t2 + 4t + 3 -> t muss ungerade sein -> t = 2t' + 1 -> 8r2b' = 8I + 4t'2 + 4t' + 1 + 8t' + 4 + 3 -> 8r2b' = 8I + 4t'(t' + 3) + 8.

Für r1 = 4 folgt mit b = 2b' und somit t = 4b/8 = b/2 = b'
-> 2r2t = 8I + t2 + 4t + 4 -> t muss gerade sein -> t = 2t' -> 4r2t' = 8I + 4t'2 + 8t' + 4 -> t' muss ungerade sein.
